* int -> long으로 선언

추가 : 참조에 의한 호출(call by reference)
void swap(int &a, int &b);

int main()
{
    int a= 10, b = 20;
    swap(a, b);
    return 0;
}
void swap(int &a, int &b)
{
    int temp = a;
    a = b;
    b = temp;
}
- 매개 변수에만 '&'표시. 주소 참조
- 매개 변수 선언 시 항상 자료형 선언 필요
